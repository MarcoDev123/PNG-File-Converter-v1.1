# PNG File Converter v1.1
PNG File Converter is a simple Windows tool for converting PNG images quickly and easily. It supports batch conversion, custom output folders, progress tracking, and a clean modern interface, making it useful for organizing images, preparing website graphics, and creating project-ready files with minimal effort.

Trial version includes 5 free conversions. Upgrade to Pro for unlimited conversions.
